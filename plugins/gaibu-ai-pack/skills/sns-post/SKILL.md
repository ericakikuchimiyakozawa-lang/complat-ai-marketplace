# /sns-post — SNS投稿文生成スキル

## トリガー

`/sns-post` と入力、または「Instagram投稿を作って」「LINE配信の文章を作って」「SNS投稿を〇本まとめて」と言われた時に起動する。

## 必要な情報（ヒアリング）

```
情報が不足している場合は以下を質問する：

Q1. どのSNSに投稿しますか？
    → Instagram / LINE / Googleビジネス / X（Twitter）

Q2. テーマ・内容は何ですか？
    → 例：夏のキャンペーン / 新メニュー / スタッフ紹介 / 定期投稿

Q3. 何本作りますか？
    → 例：5本 / 10本 / 月1本

Q4. ターゲットは誰ですか？
    → 例：30代女性 / ファミリー層 / 近隣住民

Q5. 口調・雰囲気は？
    → 例：親しみやすい / プロフェッショナル / 温かみのある
```

## 実行手順

### Step 1: SNS別フォーマット確認
各SNSで適切な文字数・構成にする：
- Instagram：300〜600文字・ハッシュタグ10〜30個
- LINE：100〜300文字・短く読みやすく
- Google ビジネスプロフィール：100〜300文字・地域名・業種を含める
- X（Twitter）：140文字以内・インパクト重視

### Step 2: 構成設計
```
① フック（最初の1〜2行で興味を引く）
② 本文（サービス・キャンペーン・価値を伝える）
③ CTA（「ご予約はプロフィールリンクから」「お電話：〇〇〇〇」等）
④ ハッシュタグ（Instagram）
```

### Step 3: 出力

**Instagram例：**
```
【投稿案 1/5】

✨〇〇〇〇〇〇〇〇（フック）

〇〇〇〇〇〇〇〇〇〇〇〇〇〇
〇〇〇〇〇〇〇〇〇〇

▶ ご予約はプロフィールのリンクから
▶ お電話でも受付中：〇〇〇-〇〇〇〇

#〇〇市 #〇〇サロン #〇〇 #〇〇 #〇〇
```

**LINE例：**
```
〇〇さん、こんにちは！

〇月〇日まで、〇〇キャンペーン開催中🎉
〇〇〇〇〇〇〇〇〇〇〇〇

ご予約はこちら → [URL]
```

## ハッシュタグ選定基準

| カテゴリ | 例 |
|---|---|
| 地域系 | #〇〇市 #〇〇駅 #〇〇区 |
| 業種系 | #美容室 #カフェ #整体 |
| サービス系 | #ヘアカラー #ランチ #肩こり |
| 季節・トレンド | #夏 #2026年夏 |
| ベネフィット系 | #癒し #綺麗になれる |

## 品質基準

- フックは「数字」「驚き」「問いかけ」のいずれかを使う
- CTAは必ず1つ以上入れる
- ハッシュタグは地域名を必ず含める（集客に直結）
