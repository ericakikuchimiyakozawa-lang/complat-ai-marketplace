# /job-posting — 求人原稿生成スキル

## トリガー

`/job-posting` と入力、または「求人を出したい」「採用原稿を作って」「Indeed・タウンワーク用に書いて」と言われた時に起動する。

## 必要な情報（ヒアリング）

```
情報が不足している場合は以下を質問する：

【必須】
Q1. 募集職種は何ですか？（例：美容師スタイリスト / ホールスタッフ / 受付）
Q2. 雇用形態は？（正社員 / パート・アルバイト / 業務委託）
Q3. 給与・時給は？（例：月給25万円〜 / 時給1,200円〜）
Q4. 勤務地の市区町村は？（例：京都市中京区）
Q5. 週何日・何時間程度の勤務ですか？

【任意だが精度が上がる】
Q6. 求める人物像は？（例：明るい・丁寧・コツコツ型・未経験歓迎）
Q7. お店・会社の特徴・強みは？（例：育休100%・研修充実・スタッフ定着率高い）
Q8. 他社との違いや自慢できる点は？
Q9. 掲載媒体（Indeed / タウンワーク / Wantedly / 自社HP 等）
```

## 実行手順

### Step 1: ターゲット求職者を定義
- 経験者向け / 未経験歓迎 / ブランク歓迎
- フルタイム / 主婦・学生 / Wワーク
- キャリアアップ重視 / 安定重視 / 働きやすさ重視

### Step 2: 差別化ポイントを抽出
「なぜ他社ではなくここで働くべきか」を1〜3点に絞る

### Step 3: 媒体別フォーマットで出力

**Indeed・タウンワーク（求人検索型）向け：**
```
仕事名（タイトル）：〇〇〇〇〇募集｜〇〇市・〇〇〇〇

仕事内容：
〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇

求める人材：
・〇〇〇〇（未経験歓迎）
・〇〇〇〇
・〇〇〇〇の方を歓迎します

給与：時給 〇,〇〇〇円〜（経験・スキルにより優遇）
待遇：交通費全額支給 / 社会保険完備 / 〇〇手当

勤務時間：〇:〇〇〜〇:〇〇（シフト制）週〇日〜OK
勤務地：〇〇市〇〇区

職場の雰囲気：
〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇
```

**Wantedly（ストーリー型）向け：**
```
タイトル：〇〇〇〇〇〇な人を求めています

こんな職場です：
〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇

こんなことをお任せします：
・〇〇〇〇
・〇〇〇〇

こんな人と働きたい：
〇〇〇〇〇〇〇〇〇〇〇〇〇〇

なぜ私たちはこの仕事をしているか：
〇〇〇〇〇〇〇〇〇〇〇〇〇〇
```

## 採用率を上げるポイント（自動で反映）

- タイトルに職種・地域名・「未経験OK」等の検索ワードを入れる
- 給与は「〇〇〇〇円〜」と下限明示（幅が広すぎると信頼を失う）
- 「あなたのことを歓迎している」メッセージを1箇所必ず入れる
- 応募の一歩を下げる文（「まずは見学だけでも」等）
